## 📊 Dataset
We use the **OrganAMNIST** dataset, which is a subset of the [MedMNIST collection](https://medmnist.com/). The dataset consists of 28x28 grayscale images and is automatically downloaded and processed via the `medmnist` Python package within the notebook.
